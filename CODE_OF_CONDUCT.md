# Code Of Conduct

## Expected Behavior

- Be respectful and direct
- Critique ideas, not people
- Prefer technical clarity over rhetoric
- Assume good intent, but document concerns concretely

## Unacceptable Behavior

- harassment or personal attacks
- discriminatory language
- deliberate disruption of technical discussion
- publishing private information without permission

## Enforcement

Project maintainers may remove comments, reject contributions, or restrict participation when behavior makes collaboration unsafe or unproductive.

Before the repository is made public, the maintainer should add the preferred contact path for conduct reports.
